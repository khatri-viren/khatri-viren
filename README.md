# Hi 👋 My name is Viren Khatri

## Full Stack Developer

I'm just a third-year Computer Science student who jumped into the coding world about a year ago, and man, what a ride it's been! I've been tinkering with PHP and SQL, getting my hands dirty crafting solid server-side solutions. Also, I've been having a blast with the MERN stack, weaving together some cool web apps that dance between the front and back ends. But it's not all serious business—my heart lies in making things look good on the frontend. I'm all about that sweet spot between making things pretty and functional.

- 🌍  I'm based in Pune, India
- 🖥️  See my portfolio at [My Portfolio](http://khatri-viren.github.io/)
- ✉️  You can contact me at [vk102002@gmail.com](mailto:vk102002@gmail.com)
- 🚀  I'm currently working on [Flora Care](http://flora-care-sigma.vercel.app)
- 🧠  I'm learning NextJs and Rust
- 🤝  I'm open to collaborating on any project realted to web development
- ⚡  Oh, and when I'm not glued to my keyboard, you'll catch me building random IoT devices, merging the digital and physical worlds. Always hungry to learn, I'm constantly chasing the latest trends and geeking out on emerging tech.
